# AI Reputation Judge

AI Reputation Judge is an Intelligent Contract built on GenLayer that evaluates community contributions using AI-powered consensus.

Unlike traditional smart contracts that rely on deterministic logic, this project leverages GenLayer's non-deterministic execution model to analyze subjective information and generate reputation scores.

---

## Why GenLayer?

Reputation is inherently subjective.

Traditional blockchains can verify transactions, but they cannot reliably evaluate contribution quality, educational impact, community value, or evidence credibility.

GenLayer introduces AI-powered consensus, enabling decentralized reasoning over non-deterministic information.

This project explores how reputation systems can evolve beyond simple on-chain metrics.

---

## Features

- AI-powered contribution evaluation
- Consensus-backed reputation scoring
- Evidence assessment
- Community impact analysis
- On-chain reputation storage
- Historical evaluation tracking

---

## How It Works

User submits:

- Contribution
- Category
- Evidence

The Intelligent Contract:

1. Evaluates contribution quality
2. Evaluates community impact
3. Evaluates originality
4. Evaluates evidence credibility
5. Generates a reputation score
6. Stores the result on-chain

---

## Example Input

Contribution:

Helped 30 new users avoid phishing attacks and secure their wallets.

Category:

Security

Evidence:

Community workshops and support sessions.

---

## Example Output

Score: 90

Reason:

High community impact through security workshops and support sessions.
